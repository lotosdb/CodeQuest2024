## Description

The Fronted solution to the CodeQuest2024 created by [DevTalles](https://cursos.devtalles.com).  
Our solution is based in [React](https://es.react.dev) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```
## Building

```bash
$ npm run build
```

## Running the app

```bash
# development
$ npm run preview
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
